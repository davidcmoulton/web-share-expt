Web sharing exploration

The aim is to register a web share target with a user agent (e.g. a mobile phone in this case), enabling the user to choose what to share with it via their device's standard sharing mechanism.

Tested on a physical OnePlus 8 Pro using Chrome.

Gubbins PoC

First, install the progressive web app that that will become the web share target. Come back here when you're done.

Now you can use your device's standard sharing system to share things with the share target - look for it in your device's share list. Try out the sharing tool over here.
